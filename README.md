# Ejercicio práctico de layout en Android

El objetivo de este ejercicio es elaborar el layout original de la aplicación de Guaper (pantalla perfil de usuario) con los recursos que nos ha proporcionado nuestro diseñador.

**Pistas**. Usar los componentes:

* RelativeLayout - LinearLayout
* ScrollView
* ImageView
* com.example.dmc.layoutexercise.utils.CustomStrokeTextView
* com.example.dmc.layoutexercise.utils.CustomTextView
* TextView
* ImageButton
